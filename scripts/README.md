# Development and Diagnostic Scripts

This directory contains scripts for development, testing, and debugging.

## API Testing Scripts

- `manual_fmp_integration_test.py` - Test FMP API with real network calls
- `diagnose_fmp_access.py` - Diagnose FMP API access issues
- `test_api_key.py` - Validate FMP API key
- `setup_alpha_vantage.py` - Test Alpha Vantage as alternative

## Usage

Run these scripts manually when:
- Testing API connectivity
- Debugging API issues  
- Validating new API keys
- Exploring alternative data providers

## Note

These are development tools, not part of the automated test suite.
For automated tests, see the `tst/` directory.
